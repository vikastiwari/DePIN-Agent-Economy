# System Components

## 1. Blockchain Contracts (`/contracts`)
- **ERC-8004 Registry (`ERC8004.sol`):** Handles the minting of Agent NFTs, identity tracking, and immutable reputation scoring.
- **Artemis Verifier (`ArtemisVerifier.rs`):** A Rust-based smart contract compiled to WASM and deployed on Arbitrum Stylus. It validates CP-SNARK proofs generated by off-chain agents.

## 2. Agent Network (`/agent`)
- **Orchestrator (`orchestrator.py`):** The LangGraph state machine that manages the agent's logic loop (reasoning, tool selection, data fetching).
- **x402 Client (`x402_client.py`):** The module responsible for intercepting HTTP 402s and generating cryptographic payment signatures.

## 3. Compute Engine (`/compute`)
- **BlockTrain Inference (`blocktrain.py`):** The localized AI inference engine utilizing PyTorch/Hugging Face to run models in a decentralized hardware topology.
- **Artemis Prover (`prover.py`):** The off-chain Rust/Python hybrid module that generates the Zero-Knowledge proofs for the AI computation.

## 4. Sub-Systems
- **TIQE Validators (Trustless Inference Quality Evaluation):** A lightweight LLM-as-a-judge system used as an optimistic fallback to catch bad actors attempting to spoof low-quality AI outputs.
